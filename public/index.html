Welcome to Airbnc API

This API provides data about the project that you can request using http endpoints.

Example of endpoints

GET /api/properties

Example request:

https://airbnc-b0sn.onrender.com/api/properties

Example response:

        {
            "properties": [
                {
                    "property_id": 10,
                    "name": "Quaint Cottage in the Hills",
